## Why LinkedIn SEO Matters

Recruiters search LinkedIn the same way you search Google: using keywords.

**The right keywords:** Increase your chances of appearing in recruiter searches

**Wrong or missing keywords:** Make your profile virtually invisible, even if you're highly qualified

### Key Statistics
- **90% of recruiters** use keyword filters to find candidates
- Candidates with keyword-optimized profiles receive **2-3x more views** from recruiters

### Goals
1. Match recruiter search terms
2. Rank high in search results
3. Attract more recruiter messages and interview opportunities

### Headline

### Engagement

## How Recruiters Search

Recruiters enter boolean searches and filter by:
- Location
- Industry
- Experience level
- Keywords

Profiles are ranked by keyword relevance and engagement.

### Example Recruiter Search String
```
("Full Stack Developer" OR "Software Engineer") AND (React OR NodeJS OR JavaScript) AND (API OR PostgreSQL OR AWS)
```

**What this tells LinkedIn:** "Find me developers who know React, NodeJS, or JavaScript AND have experience with APIs, PostgreSQL, or AWS."

---

## LinkedIn Recruiter Tool

Recruiters use a paid platform called **LinkedIn Recruiter** with:
- Custom filters (job titles, skills, location)
- Boolean search capability
- Candidate ranking by keyword matches
- Project pipelines for outreach

**Key insight:** LinkedIn matches words in recruiter searches to words in your profile. If you use different terms or omit keywords, you won't appear in results.

---

## Mirroring: Using Recruiter Language

**Critical concept:** You must use the same words recruiters use, not just words you prefer.

### Example Mismatch
- Recruiter searches: "React Developer"
- Your profile says: "Front-end engineer specializing in UI/UX"
- Result: Your profile won't appear even if your experience matches perfectly

### How to Mirror
| Recruiter Search | Your Profile Should Include |
|-----------------|----------------------------|
| Full Stack Developer | Headline: "Full Stack Developer" |
| Software Engineer | About: "As a software engineer, I build..." |
| React, NodeJS, JavaScript | Skills: React, NodeJS, JavaScript |
| API, PostgreSQL, AWS | Experience: "Developed RESTful APIs using NodeJS and PostgreSQL, deployed to AWS" |

## How to Mirror Recruiter Searches

### Step 1: Analyze Target Job Postings
- Review 3-5 postings of your dream role
- Highlight repeating terms (React, Agile, API integration, AWS)
- Note job title variations (Software Engineer, Full Stack Developer, Front-End Engineer)

### Step 2: Embed Keywords Across Your Profile
- **Headline:** Use target title + top 3 skills
- **About:** Use phrases recruiters type
- **Experience:** Tie each project/task to target keywords
- **Skills:** Add every hard skill listed in multiple job postings

### Step 3: Match Variations
Use multiple forms of the same concept:
- API and REST API
- ReactJS and React
- AWS and Amazon Web Services
- SQL and PostgreSQL

---

## Avoid Keyword Stuffing

**Don't:** Cram a list of terms without context

**Weak:** React, NodeJS, PostgreSQL, AWS, API, Agile

**Strong:** Developed full stack applications using React and NodeJS. Managed PostgreSQL databases and deployed to AWS using CI/CD pipelines.

Always use keywords in real sentences that show how you apply the skill.

---

## Applying Keywords Strategically

### Headline
Use your target title and top 3 skills.

### About Section
"I specialize in building full stack applications using React, NodeJS, and PostgreSQL. Experienced in agile workflows and API development."

### Experience Section
Add technical achievements with keywords tied to outcomes:
"Built RESTful APIs with NodeJS that reduced data load times by 40%."

### Skills Section
- Add all relevant hard skills
- Arrange top 10 most important skills first

### Synonyms
Recruiters may search different titles. Include variations:
- Full Stack Developer, Software Engineer, Web Developer
- React Developer, Front-End Engineer, UI Developer

## LinkedIn Favors Active Profiles

Boost visibility through engagement:
- Post at least once a week using relevant keywords
- Example: "Excited to finish my React app today"
- Comment on posts using industry language
- Follow companies and hashtags that reflect target roles

## Following Companies Matters

When you **follow a company**, your interest shows up in recruiter searches. This signals to recruiters that you're interested in their organization.
